# RedCanvas

RedCanvas ist ein kollaboratives Community-Board mit zwei Bereichen:
- Threads und Posts für Diskussionen
- eine gemeinsame Pixel-Leinwand im Stil von r/place

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma
- SQLite
- Zod

## Setup
npm install
npx prisma migrate dev
npx prisma generate
npm run dev
